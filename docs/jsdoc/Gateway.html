Class: Gateway

Gateway(hostname, port, pathname, timeout)

Gateway to communicate with agents from Java classes. Only agents in a master or slave container can be accessed using this gateway.

Constructor

new Gateway(hostname, port, pathname, timeout)

Creates a gateway connecting to a specified master container over Websockets.
Parameters:
Name Type Default Description
hostname string hostname of the master container to connect to
port int port of the master container to connect to
pathname string /ws/ path of the master container to connect to
timeout int 1000 timeout for fjage level messages
